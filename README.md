# Pokedex CLI

This project uses a Read, Evaluate, Print, Loop (REPL) to give the user an interactive Pokemon themed command line interface.

### Usage:
**explore \<location-name>**: Lists the Pokemon in the specified location \
**catch \<pokemon-name>**: Attempt to catch the specified Pokemon \
**help**: Displays a help message \
**inspect \<pokemon-name>**: List the stats of the specified Pokemon \
**map**: Get the next page of locations \
**mapb**: Get the previous page of locations \  
**pokdex**: Lists all your caught Pokemon \
**exit**: Exit the Pokedex \
